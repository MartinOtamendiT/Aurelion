# Idea para proyecto de ML

* Objetivo: Realizar la predicción de la cantidad del total de ventas de Julio del 2024 a Diciembre del 2025 utilizando un modelo de ML con datos históricos del mercado de Argentina.

1. Generar una nueva base de datos tomando como base las tendencias de ventas totales de la tienda Aurelion de
    Enero a Junio de 2024, y una base de datos del total de ventas histórico en el mercado argentino. Este paso lo realizaremos con ayuda de IA.
    * Tomar en cuenta fiestas, estaciones,  eventos (Black Friday, rebajas de Invierno), etc.
    * Para los clientes agregar edad (random), género (con IA de acuerdo a los nombres), compras al mes,
    antigüedad (para niveles de fidelidad).

2. Una vez generada la nueva base de datos, utilizaremos dicha base para el entrenamiento de un modelo de regresión (lineal múltiple, Random Forest y XGBoost), y la predicción de los datos de Julio a Diciembre 2024 y Enero a Diciembre 2025.
3. ¿Qué tipos de clientes son más propensos a comprar en cierta temporada? - Servirá para realizar campaña.
